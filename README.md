## 📋 نظرة عامة

**إنجاز** هو تطبيق ويب باللغة العربية (RTL) مبني على Streamlit لتحليل بيانات إكمال التقييمات الأسبوعية من ملفات Excel المُصدّرة من نظام إدارة التعلم في قطر.

### ✨ المميزات الرئيسية

- ✅ **واجهة عربية كاملة (RTL)** مع خط Cairo الاحترافي
- 📊 **تحليل تلقائي** لملفات Excel متعددة
- 🎯 **تصنيف ذكي** إلى 6 مستويات أداء
- 💡 **توصيات مهنية** جاهزة بالعربية
- 📈 **رسوم بيانية تفاعلية** مع Plotly
- 📥 **تصدير إلى Excel** مع تنسيق عربي كامل
- 🎨 **ألوان قطر الرسمية** (Maroon & Gold)

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية

- Python 3.11 أو أحدث
- pip3
- Git (للاستنساخ من GitHub)

### التثبيت السريع

```bash
# استنساخ المشروع
git clone https://github.com/YOUR-USERNAME/enjaz.git
cd enjaz

# تثبيت المتطلبات
pip3 install -r requirements.txt

# تشغيل التطبيق
streamlit run app.py
```

### الوصول إلى التطبيق

افتح المتصفح على: **http://localhost:8501**

## 📊 كيفية الاستخدام

### 1. رفع الملفات

من القائمة الجانبية، قم برفع ملف Excel واحد أو أكثر. كل ملف يجب أن يحتوي على:

- **الصف 1**: عناوين التقييمات
- **الصف 3**: تواريخ الاستحقاق
- **العمود A**: أسماء الطلاب
- **الأعمدة H+**: بيانات التقييمات

### 2. استعراض التقارير

اختر من 4 أنواع عرض:

#### 📊 لوحة المعلومات
- مؤشرات الأداء الرئيسية (KPIs)
- توزيع الفئات
- أفضل وأضعف المواد
- جدول تفصيلي للمواد

#### 📚 تقرير الصف/المادة
- إحصائيات الصف
- قائمة الطلاب مع التصنيفات
- التوصيات المهنية
- الطلاب المتميزون والذين يحتاجون اهتمام

#### 👤 ملف الطالب
- الأداء الفردي عبر جميع المواد
- رسوم بيانية تفاعلية
- التوصيات الشخصية

#### 📥 التقارير والتصدير
- تصدير التقرير الإجمالي
- تصدير تقارير المواد/الصفوف
- تنسيق Excel احترافي

### 3. التوصيات

النظام يولد توصيات مهنية تلقائياً تتضمن دائماً:
- ✅ تذكير الطلاب بحل التقييمات
- ✅ رقمنة استراتيجية الصفوف المقلوبة
- ✅ التواصل مع أولياء الأمور (للفئات الأدنى)
